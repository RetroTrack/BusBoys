# BusBoys
Dit is het project van Alex, Bart, Bram en Nicky. We maken in een paar weken een digital twin van een automone rijdende bus. 

## Installatie
Op deze pagina staat informatie over het installeren van deze github repo.
1. Open de unity hub druk en druk rechtsboven op Add.
2. Kies voor add from repository.
3. Selecteer onder repository RetroTrack/BusBoys en sla deze op waar jij zelf wil.
Hou de verschillende branches en commits goed bij.
